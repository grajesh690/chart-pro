# ChartPro Workspace

A modern, single-file interactive chart editor built with pure HTML, CSS, and JavaScript.

ChartPro Workspace provides a highly customizable visual charting experience with support for multiple chart types, editable overlays, export capabilities, workspace tabs, KPI cards, presets, and advanced styling controls — all without any external framework dependencies.

---

## ✨ Features

### 📊 Multiple Chart Types
- Line Charts
- Bar Charts
- Area Charts
- Scatter Charts
- Step Charts
- Horizontal Progress Charts

### 🎨 Rich Customization
- Fully editable chart titles and subtitles
- Axis labels and unit formatting
- Dynamic color customization
- Font styling controls
- Grid styling
- Smooth curves and area fills
- Badge and achievement callouts

### 📍 Interactive Data Editing
- Add / remove data points dynamically
- Editable labels and values
- Individual point color customization
- KPI card management

### 📐 Plateau Overlays
- Draggable and resizable plateau regions
- Custom fill and border colors
- Adjustable opacity
- Flexible label positioning

### 🗂 Workspace Tabs
- Multiple chart workspaces
- Independent chart state per tab
- Quick switching between charts

### 💾 Presets & Import/Export
- Save reusable presets
- Export/import chart JSON
- Import older ChartPro HTML files
- PNG export
- SVG export
- Crop and scale export support

### ⚡ Lightweight
- No framework dependencies
- Runs directly in the browser
- Single HTML file deployment

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/chartpro-workspace.git
cd chartpro-workspace
```

### 2. Open the Application

Simply open:

```bash
index.html
```

in your browser.

No installation or build step required.
